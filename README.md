## 环境变量说明
| 变量名      | 默认值         | 说明                                         | 是否必填 |
|-------------|----------------|----------------------------------------------|-----------|
| `FF_PORT`  | `8080`    | novnc开放端口                                | ❌ 否        |
| `FF_PASS`  |     | Xvnc连接密码                               | ❌ 否        |
| `FF_MAG`  | `0`         | 当设置为1时，启动浏览器备份管理                 | ❌ 否        |
| `FFMAG_PASS` | `ChangeMe123!@#`| 浏览器备份管理的密码                    | ❌ 否      |
